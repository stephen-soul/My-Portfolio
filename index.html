ABOUT ME

I'm A Software Developer

My name is Stephen Fetinko and I'm a software developer currently attending Nova Scotia Community College for a degree in programming. Programming for me has always been a passion of mine and its existed ever since I can remember. It started from playing around with HTML to edit home pages, and eventually it led me to learning more about the games I loved and how they were made. My passion had fuel added to the fire at the end of high school when I took a class that delved into Java and from there, I pursued learning everything possible about making anything from desktop/mobile development to game development.

Stephen Fetinko


Finding out how the world ticks was always something that interested me and figuring out complicated puzzles is always an enjoyable experience. There's not a whole lot more that can be said about me. I code a lot, I build computers and I drink a bunch of coffee. But I'm a hard worker who never leaves a task undone, and I make sure that when I'm on a team I do everything possible to make everybody's job easier. I love learning new things and if I find a website that's done a certain way or an application that's done a unique way then I go out of my way to ensure I figure out how it works so I can learn more and constantly improve. As an example, this site was built off of a W3.CSS Template with a bootswatch theme that I worked to customize and make my own.


My programming career has been and will be a journey. I've learned an incredible amount from my teachers and my peers and I look forward to learning more and more about how the world works. At the end of the day, I'm a software developer, and it's a pleasure to meet you.
